# English Level Test

O English Level Test é uma aplicação web para realização de testes de nivelamento de inglês.

O sistema permite que o aluno informe seus dados, responda às perguntas e receba seu resultado com nível de proficiência e desempenho por categoria. Também possui uma área administrativa protegida para cadastrar, consultar, editar e excluir categorias, perguntas e alternativas.

## Hospedagem

A aplicação está hospedada no Render:

- Teste: https://english-level-test.onrender.com
- Área administrativa: https://english-level-test.onrender.com/admin

O banco de dados PostgreSQL está hospedado no Supabase.
